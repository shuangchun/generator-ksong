# generator-ksong

How to install

``` 
// step 1 install yeoman
npm install yo

// step 2 install this package
npm install -g generator-ksong

// step 3 yo genertaor this project
yo ksong [app-name]

``` 
If after step3 has some error,
you can use command `sudo npm install` and `bower install`
